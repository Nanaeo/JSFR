# JSFR
遵循着一次开发处处乱跑的准则,只要你拥有nodejs那么就可以运行此工具来管理你的友情链接。
# 流程
FrameLoad -> EventHandle -> CallApi
# 插件分类
Platform 操作输入 信息回馈

Storage  数据持久化

Bulid    内容生成生产

Deploy   数据推送与部署

Other    其它附加插件 如测速 自动化审核
